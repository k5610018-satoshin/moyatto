# モヤット — 小学生向けリフレーミング練習アプリ

「モヤット」は、もやもや雲のキャラクター **モヤット** のなやみに、子どもが「ちがう見方」を教えてあげることで、リフレーミング（見方の切り替え）を練習できる授業用アプリです。小学5年生の教室での実践から生まれました。

**配布サイト**: https://k5610018-satoshin.github.io/moyatto/

## 2つの使い方

| | ① GAS版（フル機能） | ② Gemini Gem版（かんたん） |
|---|---|---|
| 準備 | Google Apps Script にコピーしてデプロイ（15分） | プロンプトを貼るだけ（2分） |
| 児童の利用 | URLを開くだけ（ログイン不要にできる） | Geminiにログインが必要 |
| なやみカード | 49種（できごと系27＋短所・苦手系22） | 同じ49種をチャットで提示 |
| AI判定 | Gemini APIで厳密判定（浅い言い換えは通さない） | Gemの指示で同等のルールを再現 |
| きもちメーター | ビジュアル表示 | 絵文字（☁️→🌤️→☀️）で表現 |
| クラス機能 | セッション・履歴・みんなの言いかえ・ベスト3投票・先生画面 | なし（1人1会話） |
| 費用 | Gemini APIキー（無料枠あり） | 無料（Geminiアカウント） |

## ① GAS版のセットアップ

**いちばん簡単な方法（約5分・コピペ不要）**: [配布プロジェクトを開く](https://script.google.com/d/1z31SLzxlT6xVxuL_vfOCMmU-GczQuKrDO75Ez2sMbbQVjAeYtHNqDlTt/edit) → 上部の「コピーを作成」→ ウェブアプリとしてデプロイ（実行=自分／アクセス=全員）→ URLに `?admin=setup` を付けて開くと、**画面内のウィザードが残りの設定（合言葉→AIキー→テスト→セッション作成）を1本道で案内**します。

手動で構築したい場合は `src/` の中身を貼り付けます。詳しくは [SETUP.md](SETUP.md) と[配布サイト](https://k5610018-satoshin.github.io/moyatto/)を見てください。

- Gemini APIキーはソースに書かず、画面から設定します（スクリプトプロパティに保存）。使用モデルは Gemini 3.1 Flash-Lite。
- 授業用スライド（PowerPoint）も配布サイトからダウンロードできます。

## ② Gem版のセットアップ

1. [gem/moyatto-gem-prompt.txt](gem/moyatto-gem-prompt.txt) の全文をコピー
2. [gemini.google.com](https://gemini.google.com) → 左メニュー「Gem マネージャー」→「Gem を作成」
3. 名前に「モヤットをたすけよう」、カスタム指示にプロンプトを貼り付けて保存

作成したGemは、Gem マネージャーの共有メニューから同僚の先生にリンク共有もできます。

## 大切にしていること（安全設計）

- リフレーミングは **「我慢の練習」ではありません**。困ったままなら先生や友だちに相談してよいことを、アプリ内でも授業でも必ず伝えます。
- 浅い受容（「気にしなくていい」）、無理なポジティブ変換、相手の不適切な行為の正当化、自己責任化は「有効な見方」として扱いません。
- いじめ・暴力などの重い内容はゲーム化せず、大人への相談につなぎます。
